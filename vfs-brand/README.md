# VFS brand lookbook

Three brand directions for **vfs.dev**, each a self-contained world: logo, type, color, voice, values, and sample UI.

```bash
bun install
bun dev
# open http://localhost:3000
```

## The three directions

1. **DEPOT** — industrial / utilitarian. VFS as operational primitive. Stencil as shipping manifest.
2. **ARCHIVE** — editorial / archival. VFS as universal index. Stencil as Bell Labs monograph.
3. **PROTOCOL** — protocol-first / agent-era. VFS as substrate. Stencil as callsign.

Each direction is a real position with real trade-offs, not a color swap. Read the *Positioning* and *Trade-off* blocks on each page.
